# 🌊 AI-Based IoT for Water Quality and Contamination Control

## 📌 Overview

This project is an IoT-based smart system that monitors water quality in real time and detects contamination using machine learning algorithms. It uses sensors connected to an ESP32 microcontroller to collect data and send it to cloud platforms for analysis and visualization.

---

## 🎯 Objectives

* Monitor water quality parameters continuously
* Detect contamination using AI models
* Provide real-time alerts to users
* Ensure safe and clean water monitoring

---

## 🚀 Features

* 📡 Real-time water quality monitoring
* 🤖 AI-based contamination detection
* 🌐 Cloud data storage and visualization
* 🔔 Instant alerts via Telegram
* 📱 Remote monitoring using IoT platforms

---

## 🛠️ Hardware Components

* ESP32 Microcontroller
* pH Sensor
* Temperature Sensor
* Ultrasonic Sensor (Water Level)
* TDS Sensor (Total Dissolved Solids)

---

## 💻 Software & Tools

* Arduino IDE
* Python

---

* Random Forest

* scikit-learn

---

## ☁️ IoT Platforms

* ThingSpeak – Data visualization
* Blynk – Remote monitoring and control

---

## 🔗 Communication Protocol

* MQTT (Message Queuing Telemetry Transport)

---

## 🔔 Alert System

* Telegram Bot integration
* Sends notifications when water quality exceeds safe limits

---

## ⚙️ Working Process

1. Sensors collect water quality data
2. ESP32 processes the data
3. Data is sent to cloud via MQTT
4. Machine learning model analyzes contamination
5. Results are displayed on dashboards
6. Alerts are sent via Telegram

---

## ▶️ How to Run the Project

### 🔹 Step 1: Hardware Setup

Connect all sensors to ESP32 properly.

### 🔹 Step 2: Upload Code

Upload Arduino code using Arduino IDE.

### 🔹 Step 3: Run Python Script

```bash
python main.py
```

### 🔹 Step 4: Monitor Output

* Check data on ThingSpeak/Blynk
* Receive alerts on Telegram
